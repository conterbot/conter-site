Sobre el proyecto

Es un proyecto OpenSource llevado a cabo por alumnos de Penguin Academy, cuyo objetivo es contribuir a que los usuarios puedan llegar a su destino a tiempo, proporcionando información acerca del estado del tráfico en determinadas zonas.

Python

Se trata de un lenguaje de programación multiparadigma, ya que soporta parcialmente la orientación a objetos y programación imperativa, además es un lenguaje interpretado, dinámico y multiplataforma; además posee una licencia de código abierto, denominada Python Software Foundation License. Este proyecto está desarrollado en el Lenguaje Python.

Machine Learning

El Machine Learning es una disciplina del campo de la Inteligencia Artificial que, a través de algoritmos, dota a los ordenadores de la capacidad de identificar patrones en datos masivos y elaborar predicciones (análisis predictivo). Este aprendizaje permite a los computadores realizar tareas específicas de forma autónoma, es decir, sin necesidad de ser programados. ¡Lo utilizamos para identificar y contar los vehículos!

Telegram Bot

Es la interfaz de programación del servicio de mensajería de la aplicación "Telegram" orientada a los bots, realiza tareas repetitivas, predefinidas y automatizadas. ¡En nuestro proyecto lo utilizamos para informar al usuario el estado del tráfico!
